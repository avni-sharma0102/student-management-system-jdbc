# Student Management System (Java + JDBC + MySQL)

## Project Overview
This is a console-based Student Management System built using Java, JDBC, and MySQL. It performs CRUD operations with database connectivity.

## Features
- Add Student
- View Students
- Update Student
- Delete Student

## Tech Stack
- Java
- JDBC
- MySQL

## How to Run
1. Import project in IDE
2. Setup MySQL database
3. Update DB credentials in code
4. Run main file
